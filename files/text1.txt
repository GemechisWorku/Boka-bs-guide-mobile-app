YAADA GARAA
Kitaaba Qulqulluu keessatti termiin ‘yaada garaa’ jedhu  kallattiinis ta'u al-kallattiin yeroo baayyee barreeffamee argina. Keessumaa “yaada garaa geeddarachuu,” “qalbii diddiirrachuu” fa’aa yeroo hunduu dhageenya; arginas.  Ka'umsaaf akka tolutti luqqisiiwwan muraasa irratti haasa'uuf nuu tolu jedhee yaade kanaa gaditti kaa'uuf yaaleera.

Fak 4:24 => “Wanti jireenyaaf barbaachisu yaada garaa keessaa waan madduuf yaada kee 			ba’eessa  godhii eeggadhu.” 
Ibr 13:18 => “Waan gaariif waan hawwinuuf yaadni garaa keenyaa nu hin ceepha’u.”
Rom 12:8 => “Inni namaaf kennu garaa qulqulluudhaan haa kennu.”
Mat 6:33 fi Luq 12:31 => “Duraan dursitanii yaada keessan mootummaa waaqayyoo fi 		qajeelummaa isaa barbaaduutti hidhaa! Kana malees kan barbaachisu 	hundinuu immoo isiniif in kennama.”
Qol 3:5 => “Egaa yaada keessan wanta lafa irra jirutti isa hidhu, halalummaa fi xuraa’ummaa, 		kajeellaa fi hawwa hamaa, sassata isa waaqayyoolii tolfaman waaqessuutti 			lakkaa’amus, of keessaa ajjeesaa.”
Marq 7:15 => “Alaa kan namatti galu tokkollee danda’ee xuraa’aa nama hin godhu, wanti yaada 		garaa namaa keessaa ba’u garuu xuraa’aa nama godha.”
Marq 7:20 => “Yaadni hamaan hundinuu keessa yaada garaa namootaatii in ba’a; isaan immoo halalummaa, hanna……..dagattummaa.”

Efe 4: 17-24 => jireenya haaraa karaa Kiristoos argamu akkatti jiraatamu kan ibsu
Rom 1:18-32 => yaadaa waa’ee hin baafne duukaa bu’uun wantoota inni fidu kan nu hubachiisu
1Xim 6:10-11, Ibr 9: 14 
Kanneen erga dubbifattanii irratti yaaddanii ykn ‘meditate’ gootanii booda qabxiilee armaan gadii walbira qabdanii hariiroo fi garaagarummaa isaanii ilaaltanii akka dhuftan jaalala gooftaatiin isinan gaafadha!
